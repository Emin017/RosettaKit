# RosettaKit

[![CI](https://github.com/Emin017/RosettaKit/actions/workflows/ci.yml/badge.svg)](https://github.com/Emin017/RosettaKit/actions/workflows/ci.yml)
[![PyPI Publish](https://github.com/Emin017/RosettaKit/actions/workflows/publish.yml/badge.svg)](https://github.com/Emin017/RosettaKit/actions/workflows/publish.yml)
[![GitHub Release](https://img.shields.io/github/v/release/Emin017/RosettaKit?label=GitHub%20release)](https://github.com/Emin017/RosettaKit/releases)
[![PyPI](https://img.shields.io/pypi/v/rosettakit.svg)](https://pypi.org/project/rosettakit/)

RosettaKit is a small Python library for building typed EDA script documents and
rendering them into tool-facing text such as Tcl fragments and command files.

RosettaKit only builds script text. It does not parse Tcl, execute EDA tools,
manage subprocesses, or model a host workflow.

RosettaKit is published on PyPI: <https://pypi.org/project/rosettakit/>.

## Install

```bash
uv add rosettakit
```

or:

```bash
pip install rosettakit
```

## Quick Start

Use `rosettakit.tcl` for Tcl fragments:

```python
from rosettakit import tcl

script = tcl.Script()
script.comment("Auto-generated by RosettaKit")
script.set("top_module", tcl.word("example_core"))
script.set("clock_mhz", 500)
script.set_path("work_dir", "build outputs/run 1")
script.set_list("library_paths", ["libs/typical.lib", "libs/slow corner.lib"])

with script.if_not(tcl.file_isdirectory(tcl.var("work_dir"))):
    script.file_mkdir(tcl.var("work_dir"))

text = script.build()
```

Use `rosettakit.cmdfile` for flag and option files:

```python
from rosettakit import cmdfile

cmd = cmdfile.CommandFile(prefix="-")
cmd.flag("verbose")
cmd.option("top", "example_core")
cmd.option("threads", 8)
cmd.option("input", "build outputs/design.v", value_type=cmdfile.ValueType.PATH)
cmd.options(
    "include",
    ["config/default.cfg", "config/local override.cfg"],
    value_type=cmdfile.ValueType.PATH,
)

text = cmd.build()
```

Both document types preserve insertion order and provide `validate()` for
diagnostics. Raw text is an explicit escape hatch; builds reject raw content
unless `allow_unsafe_raw=True` is passed.

## API Stability

RosettaKit 0.2.0 treats the public `rosettakit.tcl`,
`rosettakit.cmdfile`, `rosettakit.diagnostics`, and `rosettakit.errors` APIs as
stable. Generated text is part of that compatibility surface, including
quoting, indentation, line ordering, and the default raw-content policy.

## Examples

First-party examples use only RosettaKit and Python standard-library modules:

- [Tcl fragment generation](examples/yosys_global_var.py)
- [Flag and option command file generation](examples/sizer_cmd_file.py)
- [Env-style command file generation](examples/sizer_env_file.py)

```bash
uv run python examples/yosys_global_var.py
uv run python examples/sizer_cmd_file.py
uv run python examples/sizer_env_file.py
```

They write generated output under ignored paths:

```text
examples/out/yosys/global_var.tcl
examples/out/sizer/design.cmd_file
examples/out/sizer/design.env_file
```

You can also smoke-test the generated Tcl with a local Tcl shell:

```bash
uv run python examples/yosys_global_var.py
tclsh examples/out/yosys/global_var.tcl
```
