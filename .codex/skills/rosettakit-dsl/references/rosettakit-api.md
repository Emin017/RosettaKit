# RosettaKit API Reference

This reference summarizes RosettaKit 1.1.0 as observed in the local repository.
Verify against the active package when working in a repo, because generated text
is part of RosettaKit's compatibility surface.

## Core Boundary

RosettaKit is a typed Python text-generation DSL for EDA artifacts. It builds
Tcl fragments and command files. It does not parse Tcl, run EDA tools, manage a
host workflow, or own subprocess execution.

Public modules:

- `rosettakit.tcl`
- `rosettakit.cmdfile`
- `rosettakit.diagnostics.Diagnostic`
- `rosettakit.errors`

## Tcl API

Primary types and helpers:

- `tcl.Script`
- `tcl.TclBuilder`
- `tcl.Scalar`, `tcl.PathValue`, `tcl.ListValue`, `tcl.VarRef`, `tcl.Expr`,
  `tcl.CommandSubstitution`, `tcl.Raw`
- `tcl.Condition`, `tcl.Comment`, `tcl.BlankLine`, `tcl.Set`, `tcl.Command`,
  `tcl.If`, `tcl.RawLine`
- `tcl.word`, `tcl.path`, `tcl.list_value`, `tcl.var`, `tcl.expr`, `tcl.call`,
  `tcl.raw`, `tcl.file_isdirectory`

Example:

```python
from rosettakit import tcl

script = tcl.Script()
script.comment("Auto-generated by RosettaKit")
script.set("top_design", tcl.word("gcd_core"))
script.set("clk_freq_mhz", 500)
script.set_path("final_netlist_file", "build out/gcd final.v")
script.set_list("lib_list", ["libs/fast corner.lib", "libs/slow.lib"])
script.set_expr("clk_period_ps", "1000000.0 / $clk_freq_mhz")
script.set_path("tmp_dir", "build out/tmp")
script.file_mkdir(tcl.var("tmp_dir"))
script.set_path("synth_stat_json", "reports/synth stat.json")
script.set("stat_dir", tcl.call("file", "dirname", tcl.var("synth_stat_json")))

with script.if_not(tcl.file_isdirectory(tcl.var("stat_dir"))):
    script.file_mkdir(tcl.var("stat_dir"))

text = script.build()
```

Expected shape:

```tcl
# Auto-generated by RosettaKit
set top_design gcd_core
set clk_freq_mhz 500
set final_netlist_file {build out/gcd final.v}
set lib_list [list {libs/fast corner.lib} libs/slow.lib]
set clk_period_ps [expr {1000000.0 / $clk_freq_mhz}]
set tmp_dir {build out/tmp}
file mkdir $tmp_dir
set synth_stat_json {reports/synth stat.json}
set stat_dir [file dirname $synth_stat_json]
if {!([file isdirectory $stat_dir])} {
    file mkdir $stat_dir
}
```

Tcl rules:

- `script.set(...)` treats values as one scalar word.
- `script.set_list(...)` must be used for Tcl lists; passing `tcl.list_value`
  through scalar `set` reports `list-through-scalar-api`.
- `script.set_path(...)` gives path-aware diagnostics such as `empty-path` and
  nonblocking `quoted-path`.
- `script.comment(...)` emits one commented line per input line and escapes
  braces/backslashes for safe use inside blocks.
- `script.if_not(condition)` negates the whole condition as `!({condition})`.
- Raw values and raw lines report `unsafe-raw`; `build()` rejects them unless
  called with `allow_unsafe_raw=True`.

## Command-File API

Primary types and helpers:

- `cmdfile.CommandFile`
- `cmdfile.CommandFileBuilder`
- `cmdfile.ValueType`
- `cmdfile.ValueQuoting`
- `cmdfile.CommandFileDialect`
- `cmdfile.TCL_WORD_DIALECT`
- `cmdfile.PLAIN_DIALECT`
- `cmdfile.Comment`, `cmdfile.BlankLine`, `cmdfile.Flag`, `cmdfile.Option`,
  `cmdfile.RawLine`

Default Tcl-word dialect example:

```python
from rosettakit import cmdfile

cmd = cmdfile.CommandFile(prefix="-")
cmd.flag("useOpenSTA")
cmd.option("top", "gcd_core")
cmd.option("def", "build out/input.def", value_type=cmdfile.ValueType.PATH)
cmd.option("v", "", value_type=cmdfile.ValueType.PATH, omit_empty=True)
cmd.option("sdc", "constraints/main clock.sdc", value_type=cmdfile.ValueType.PATH)
cmd.options("lef", ["tech/sky130.lef", "macro lef/sram.lef"], value_type=cmdfile.ValueType.PATH)
text = cmd.build()
```

Expected shape:

```text
-useOpenSTA
-top gcd_core
-def {build out/input.def}
-sdc {constraints/main clock.sdc}
-lef tech/sky130.lef
-lef {macro lef/sram.lef}
```

Plain dialect example:

```python
from rosettakit import cmdfile

cmd = cmdfile.CommandFile(prefix="-", dialect=cmdfile.PLAIN_DIALECT)
cmd.flag("useOpenSTA")
cmd.option("top", "gcd_core")
cmd.option("def", "build/input.def", value_type=cmdfile.ValueType.PATH)
cmd.options("lib", ["lib/slow.lib", "lib/fast.lib"], value_type=cmdfile.ValueType.PATH)
text = cmd.build()
```

Env-file style example:

```python
from rosettakit import cmdfile

env = cmdfile.CommandFile(prefix="")
env.comment("Sizer environment")
env.option("set_db", "design_process_node 130")
env.blank_line()
env.option("set_db", "design_netlist_file build/gcd.v")
text = env.build()
```

Command-file rules:

- Values are rendered in insertion order.
- `TCL_WORD_DIALECT` quotes whitespace or Tcl-sensitive characters as one word.
- `PLAIN_DIALECT` rejects empty scalar values, whitespace, line breaks, and
  quoting-sensitive characters unless an empty value is omitted.
- Path values with whitespace in the default dialect produce a nonblocking
  `quoted-path` diagnostic and still build.
- Line breaks in option values produce a blocking `line-break-in-value`
  diagnostic.
- Raw lines report `unsafe-raw`; `build()` rejects them unless called with
  `allow_unsafe_raw=True`.

## Diagnostics And Errors

Use `document.validate()` before build when migrating or reviewing generated
artifacts. Relevant diagnostic codes include:

- `empty-variable-name`
- `empty-command-name`
- `empty-condition`
- `empty-option-name`
- `empty-path`
- `quoted-path`
- `list-through-scalar-api`
- `unsafe-raw`
- `unsupported-node`
- `unsupported-command-file-dialect`
- `line-break-in-value`
- `empty-unquoted-value`
- `unquoted-value-needs-quoting`

`document.build()` raises `ValidationError` for blocking diagnostics and
`UnsafeRawError` for raw content unless unsafe raw output is explicitly allowed.

## Verification Patterns

Use exact output assertions for generated text:

```python
assert script.build() == (
    "# Auto-generated by RosettaKit\n"
    "set top_design gcd_core\n"
)
```

For runnable Tcl fragments, smoke-test with `tclsh`:

```bash
uv run python examples/yosys_global_var.py
tclsh examples/out/yosys/global_var.tcl
```

For RosettaKit repository work, prefer:

```bash
uv sync
uv run pytest
uv run ruff check
```
